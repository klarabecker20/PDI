Input Type File

O que é um formulário em HTML? Um formulário HTML é usado para coletar a entrada do usuário. A entrada do usuário é mais frequentemente enviada a um servidor para processamento.

Veja o tipo "Input File"

O arquivo de tipo de entrada define um campo de seleção de arquivo e um botão "Procurar" para uploads de arquivos.
